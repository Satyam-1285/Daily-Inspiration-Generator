# ✨ Daily Inspiration Generator ✨

A simple, interactive website that displays a random motivational quote each time you click a button.  
Built with **HTML, CSS, and JavaScript** — all in a single file for easy hosting on GitHub Pages.

---

## 🚀 Features
- Clean, responsive design with a gradient background
- Random quote generator powered by JavaScript
- Single-file project (no external dependencies)
- Easy to customize with your own quotes or styles
